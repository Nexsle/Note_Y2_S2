# Control Structure: Switch
---
```c++
switch (num)
{
	case 1: cout << "1"; break;
	case 2: cout << "2"; break;
	case 3: cout << "3"; break;
	case 4: cout << "4"; break;
}
```


- Declare a var inside a switch
```c++
switch (int num = randint(2); num)
```
